# FocusDo 待解决问题

## 1. 筛选交互优化
**问题：** 每个 project 列都有独立的筛选 Tab（全部/进行中/今日/已完成），UI 上显挤，但功能上又需要。
**方向：** 需要一个更优雅的筛选方案，可能是全局筛选 + 列内简化，或者把筛选移到其他位置。

## 2. 已完成事项排序
**问题：** 完成的 todo 和未完成的混在一起，视觉上很乱。
**方向：** 已完成的自动沉底，或者默认隐藏（通过筛选 Tab 查看）。需要设计一个合理的排序规则。

## 3. 新增 Project 功能
**问题：** 目前三列项目是写死的，无法自定义。
**方向：** 支持新增/编辑/删除 Project，包括名称、图标、颜色、分类。
